## Step Types

Each step needs exactly one of: instruction, include_path, include_glob.

Instruction step fields:
  instruction        string   required  Plain-English instruction for the agent
  step_timeout_secs  integer  optional  Override timeout for this step
  skip               boolean  optional  Skip this step (counts as passed)
  checkpoint         string   optional  Save state after pass, restore if skipped

Include step fields:
  include_path       string   Relative path to a test file to inline
  include_glob       string   Glob pattern to inline multiple files (sorted alphabetically)
